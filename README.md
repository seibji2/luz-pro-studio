## 🚀 Cómo Usar

1. Abre el enlace de la demo
2. Sube o arrastra tu foto
3. Elige un filtro o deja que la IA sugiera
4. Ajusta a tu gusto
5. Descarga o comparte

## 📲 Instalar como App

### Android/iOS
1. Abre la app en Chrome/Safari
2. Menú → "Agregar a pantalla de inicio"
3. ¡Listo! Funciona como app nativa
